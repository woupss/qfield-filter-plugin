qfield-filter-plugin make filter in QField a reality.

With it you can select a layer and a field of your project and write 1 or many values to filter on this define layer field.

You can filter number or text and it is not case sensitive

It will highlight filtered geometries and hide others.

If you longpress the plugin icon it will delete the filter.

It is a starting project so, i hope, it will be even better in future

Installation
To install the plugin, download the plugin from the releases page and follow the plugin installation guide to install the zipped plugin in QField.
